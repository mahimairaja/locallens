# Dashboard: Search

The search page provides a visual interface for semantic search.

## Features

- Text input for search queries
- Results displayed as cards with file name, score, and chunk preview
- File type filter dropdown
- Click a result to see the full chunk text
- Results sorted by relevance score

## Usage

1. Navigate to the Search page from the sidebar
2. Type your query in the search box
3. Optionally filter by file type
4. Results appear ranked by relevance
